# 政治理论热词派🔥 部署指南

## 方式一：GitHub Pages（推荐）

### 1. 创建 GitHub 仓库

1. 登录 GitHub，点击右上角 `+` → `New repository`
2. 仓库名建议：`politics-flashcards` 或 `政治理论热词`
3. 选择 `Public`
4. 勾选 `Add a README file`（可选）
5. 点击 `Create repository`

### 2. 上传文件

**方式 A：网页上传（适合小文件）**

1. 进入刚创建的仓库
2. 点击 `Add file` → `Upload files`
3. 把以下文件和目录拖入：
   - `index.html`
   - `data-theory.js`
   - `data-qiushi.js`
   - `data-hot.js`
   - `README.md`
   - `.gitignore`
   - `templates/` 目录（可选）
   - `scripts/` 目录（可选）
4. 点击 `Commit changes`

**方式 B：Git 命令行（推荐）**

```bash
# 在项目目录初始化 Git
cd "/Users/zhaochanghe/Desktop/政治理论热词"
git init
git add .
git commit -m "初始提交：政治理论热词派"

# 关联远程仓库（替换 YOUR_USERNAME 和 YOUR_REPO）
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO.git
git branch -M main
git push -u origin main
```

### 3. 启用 GitHub Pages

1. 进入仓库首页
2. 点击 `Settings` → `Pages`（左侧菜单）
3. `Source` 选择 `Deploy from a branch`
4. `Branch` 选择 `main` 和 `/ (root)`
5. 点击 `Save`

等待 1-2 分钟，访问 `https://YOUR_USERNAME.github.io/YOUR_REPO/` 即可！

双击 `index.html` 用浏览器打开即可，无需上传。

## 数据更新

如需更新数据，使用 `scripts/` 目录下的 Python 脚本：

```bash
# 理论基础知识
python scripts/excel_to_theory_js.py 新数据.xlsx data-theory.js

# 求是
python scripts/excel_to_qiushi_js.py 新数据.xlsx data-qiushi.js

# 热点文件
python scripts/excel_to_hot_js.py 新数据.xlsx data-hot.js
```

Excel 模板见 `templates/` 目录。

---

## 注意事项

1. **数据文件大小**：`data-qiushi.js` 约 580KB，如果超过 GitHub 单文件限制（100MB），建议拆分
2. **中文文件名**：模板文件使用中文名，在 Git 中可能需要配置 `git config core.quotepath false`
3. **本地测试**：上传前建议本地双击 `index.html` 测试功能是否正常

## 常见问题

### Q: GitHub Pages 访问慢？
A: 可以使用 CDN 加速，或者部署到国内的代码托管平台（如 Gitee）

### Q: 如何更新数据后重新部署？
A: 修改 `data-*.js` 后，提交并推送到 GitHub，GitHub Pages 会自动重新部署（约 1-2 分钟）

### Q: 可以让别人直接使用吗？
A: 可以！分享你的 GitHub Pages 链接即可，例如：`https://YOUR_USERNAME.github.io/YOUR_REPO/`
