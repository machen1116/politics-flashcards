# 政治理论热词派🔥

基于翻转卡片的公务员考试政治理论学习 Web 应用。

## 功能特色

- 📚 **三大知识库**：理论基础知识、求是期刊、热点文件
- 🃏 **翻转卡片**：点击卡片翻转查看答案，模拟真实记忆卡片体验
- 🎯 **三种学习模式**：顺序学习、随机学习、收藏夹
- 📊 **学习统计**：实时显示进度、掌握率、连胜记录
- 💾 **本地保存**：学习进度和收藏自动保存在浏览器 localStorage

## 数据规模

| 分类 | 内容 | 卡片数 |
|------|------|--------|
| 理论基础知识 | 习近平新时代中国特色社会主义思想 + 马克思主义基本原理 | 372张 |
| 求是 | 2024-2026年各期期刊 | 809张 |
| 热点文件 | 政府工作报告、中央一号文件、十五五规划、中央经济工作会议 | 373张 |

## 使用方法

1. 直接用浏览器打开 `index.html` 即可使用（无需服务器）
2. 或部署到 GitHub Pages：将本仓库文件放在 `docs/` 目录或根目录即可

## 数据更新

如需更新数据，使用 `scripts/` 目录下的 Python 脚本：

```bash
# 理论基础知识
python scripts/excel_to_theory_js.py 模板文件.xlsx data-theory.js

# 求是
python scripts/excel_to_qiushi_js.py 模板文件.xlsx data-qiushi.js

# 热点文件
python scripts/excel_to_hot_js.py 模板文件.xlsx data-hot.js
```

Excel 模板见 `templates/` 目录。

## 技术栈

- 纯前端（HTML + CSS + JavaScript）
- 无框架依赖，单文件即可运行
- 数据外挂为 `.js` 文件，方便更新
